Given a [[topological space]] $$(M, \mathcal O)$$ and a set of subsets $$C\subseteq \mathcal P(M)$$ is called a cover of $$M$$ if $$\bigcup C = M$$

An open cover follows as $$C\in\mathcal O$$.

See also: [[subcover]], [[cover refinement]]
